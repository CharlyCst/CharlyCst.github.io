Projects

Zephyr 2020-2021

A small (work in progress) programming language that compiles to WebAssembly.

A lot of languages are being retro-fitted to compile down to Wasm, however they are often designed to run on a single kind of platform (Posix-like usually), that is why most of Wasm in use today expect to run on WASI and compilers tools such as Emscripten where built.

Wasm offers much more flexibility, and I think that to get the most out it runtime bindings must be a language feature rather than baked in our compilers. Zephyr is an experiment to showcase the power of abstracting runtime interfaces in the language itself.

Parrot 2020-2021

A command line tool for testing other command line tools by saving snapshots of their outputs.

End-to-end testing of error messages of the Zephyr compiler quickly became untractable, I built Parrot to help manage those tests by providing a simple CLI, a way to browse tests snapshots and to visualize diff in case the output changed, so that the final user can decide itself if it’s an improvement or a regression.

Go enclosures 2020

Worked on extending the Go compiler and runtime to support execution of closures in isolated environment as a Semester Project at EPFL.

I built a backend for LitterBox (a framework providing fast context switches between different access rights over the memory) based on the new Intel MPK extension, and then integrated it in the Go language. I demonstrated low overhead and my code was used in the final version, whose paper has been accepted to ASPLOS 2021.

Wasm memory explorer 2020

An online tool to interact with WebAssembly modules and explore their memory.

I built it while working on memory allocation for Zephyr, it turned out that coupled with browser’s built-in Wasm debugger it’s a very versatile tool. I end up using it on a regular basis when working on Wasm binaries.

Tendermock 2020

A fake Tendermint blockchain node that mimic the behavior of a full chain.

The goal of Tendermock is to be used for integration testing of the Inter BlockChain (IBC) protocol, more precisely the Rust implementation of the relayer. The node features a JsonRPC, a Websocket and a gRPC interface, is compatible with the ICS26 routing module and can generate ICS23-compatible existence proofs.

Mario RL 2020

An online reinforcement learning playground I built while following a RL class at EPFL.

Want to see the difference between Sarsa and Q-Learning for real? Give it a try!

Cloud GBA 2020

Cloud gaming but for Gameboy Advance!

A project I built over a weekend with a friend for an Hackathon. The hard part was getting the streaming right: at 30 fps a 240x160 pixels screen without compression yields 3.5 Mb/s of outbound traffic. Hackathons being hackathons, we compressed the images to JPEG and got something a bit more reasonable. Ideally one would compress a few frames together and exploit the similarity with previous screens.
